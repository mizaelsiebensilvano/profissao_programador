Box Model

Position Fixed - define que um elemento fica fixo, ou seja, colado no topo e na lateral esquerda do navegador, usando os scrolls, o objeto fica na mesma posição sempre, não mudando de lugar.
Position Sticky - define que um elemento fique colado nas bordas do navegador, assim que os scrolls são acionados, os objetos colam nas bordas do navegador.
Position Relative - define que um elemento fique a uma distância do elemento ao lado.
Position Absolute - se comporta de duas maneiras, dentro de um static ele vai levar em consideração os limites da tela, dentro de um relative ele vai levar em consideração os limites elemento pai.

Background

Background Image - permite que coloque uma imagem no fundo do HTLM.
Background Repeat - permite que a imagem não repita usando o comando no-repeat
Background Size: contain - permite que a imagem fique contida no máximo de qualidade possível. Background Size: auto - permite que a imagem fique no valor ideal.
Background Size: cover - permite que a imagem ocupe todo o espaço do bloco.
Background Position - define o posicionamento da imagem dentro do bloco usando os valores: 20%(horizontal) 20%(vertical);.

Background

Cores - utilizamos cores hexadecimal ou rgb.

Pseudo Classes & Pseudos Elementos

Hover - adiciona um novo CSS pro elemento ao passar o mouse em cima. Transition - período que a transição do CSS aconteça.
Input: disable - utilize o comando border-color: red, para input desabilitado (.input disable).

Display

Display inline - coloca dois elementos lado a lado, não é possível atribuir altura ou largura específica.
Display inline-block - funciona como inline, mas nesse caso é possível atribuir altura e largura.
Display block - adiciona uma quebra de linha em cada elemento.
Display none - é a ausência de display, fica invível no documento HTML.

Display Flex

Display Flex - sempre vai ser inserido do bloco pai, isso vai ser refletido nos posicionamentos do blocos filhos. Quando inserimos, inicialmente ele vai colocar todos os filhos lado a lado.
Flex Direction
Flex-direction: row; - define a direção dos nossos itens, row é o valor padrão.
Flex-direction: row-reverse; - inverte a ordem dos elementos e coloca do outro lado do navegador, sou seja, tudo ao contrário do modo padrão.
Flex-direction: column; - os elementos vão se comportar como colunas.
Flex-direction: column-reverse; - inverte a ordem dos elementos em colunas.
Justify-content alinha os elementos horizontalmente
Justify-content: flex-start - os elementos adotam o comportamento inicial.
Justify-content: flex-end - os elementos vão estar do outro lado inverso do inicial, sem alterar a ordem dos elementos.
Justify-content: center - deixa os elementos ao centro.
Justify-content: space-between - separa os elementos ao máximo.
Justify-content: space-around - separa os elementos ao máximo e faz espaçamentos nas laterais.
Justify-content: space-evenly - deixa todos os espaçamentos iguais desde as laterais e entre os elementos também.
Align-items alinha os elementos verticalmente
Align-items: flex-start - alinha verticalmente no topo do navegador.
Align-items: flex-end - alinha os elementos para o final do bloco pai.
Align-items: center - alinha verticalmente ao centro do bloco pai.

Display Grid

Display Grid - é inserido no bloco pai, alterando os elementos nos blocos filhos.
grid-template-columns - utilizando o comando "1fr 1fr 1fr ou repeat(3. 1fr) ou 10% 20% 70%" divide os elementos em três colunas.
grid-template-rows - utilizando o comando "repeat(2. 1fr)" divide os elementos em números de linhas.
column-gape - define a distância entre as colunas.
row-gap - define a distância entre as linhas.

Keyframes

@Keyframes from {border radius: 0} to {border-radius: 50%}, o quadrado vai virar um círculo.
animation - indica o nome da animação configurada no @Keyframes.
animation-duration - indica quanto tempo essa animação vai durar.

Media Query

Responsividade - é a adaptação de um site para diferentes dispositivos.
@media - utiliza para adaptar o site em diferentes tamanhos.
@media(min-width) ou @media(max-width) configura o tamanho mínimo da largura e o tamanho máximo da largura.
